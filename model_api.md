# الدليل الشامل لـ Google Gemini API (OSARA AI DESGINE) - Paid Tier 1

هذا الدليل يحتوي على جميع النماذج المدعومة في حسابك، مع **الاسم البرمجي الدقيق** (Exact ID)، **الحدود القصوى للطلبات (Rate Limits)**، و**أفضل استخدام لكل نموذج**.

* **RPM**: طلب في الدقيقة (Requests Per Minute)
* **TPM**: توكن في الدقيقة (Tokens Per Minute)
* **RPD**: طلب في اليوم (Requests Per Day)
